# 纯数字电路数字钟Proteus仿真设计

## 资源描述

本仓库提供了一个纯数字电路数字钟的Proteus仿真设计文件，该设计主要使用了以下数字集成电路：

- 74LS90：十进制计数器
- 555定时器：用于产生时钟信号
- 74LS161：四位二进制计数器
- 74LS48：BCD到七段显示译码器

通过这些元件的组合，实现了数字钟的基本功能，包括时、分、秒的显示与计时。
